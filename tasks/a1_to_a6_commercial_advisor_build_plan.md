# A1-A6 工程任务拆解

生成时间：2026-06-11

## A1：隐私与数据分级模型

只允许做：

- 数据分级规格
- 敏感字段识别规则
- 脱敏规则
- 外发上下文规则

## A2：私密数据保险库

允许做：

- 本地加密数据模型
- 用户画像本地存储
- 关注列表
- 项目列表
- 长期目标
- 访问日志

禁止：

- 真实券商账户
- 银行账户
- 自动同步私人数据

## A3：LLM Gateway

允许做：

- 模型配置
- 脱敏任务包
- 调用日志
- 输出审查

禁止：

- 原始敏感数据直发模型
- 无日志调用
- 无来源事实生成

## A4：市场军师 MVP

允许做：

- 市场问题识别
- 行情/新闻数据输入
- 用户关注列表关联
- 持仓摘要关联
- 简短军师判断

禁止：

- 自动交易
- 直接买卖建议
- 编造实时行情
- 接券商账户

## A5：项目军师 MVP

允许做：

- 项目状态输入
- 阶段判断
- 下一步行动
- 不要做清单

## A6：个人决策军师 MVP

允许做：

- 长期目标对账
- 行为模式分析
- 反方判断
- 机会成本分析
- 行动建议
