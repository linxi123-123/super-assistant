# Privacy Gateway Spec

## 目标

隐私网关位于用户输入、用户画像和 LLM Gateway 之间，负责确保发送给外部模型的上下文只包含必要且脱敏的信息。

## 当前策略

1. 文本脱敏
   - 手机号
   - 邮箱
   - 身份证号
   - 成本、仓位、资产、账户、金额、地址等显性敏感字段

2. 用户画像脱敏
   - 关注列表可保留标的名称和市场，因为这是军师判断所需上下文。
   - 持仓不直接暴露成本、仓位和账户细节，只转为摘要。
   - `P4_SECRET` 项不得进入建议上下文。

3. LLM 前置约束
   - 不得直接建议买入。
   - 不得直接建议卖出。
   - 不得假装实时。
   - 不得编造行情。
   - 不得泄露敏感信息。

4. 本地复核
   - LLM 或领域服务返回后，`local_judge` 检查高风险表达。
   - 缺少风险提示或不要做清单时，自动追加风控提醒。

## 加密边界

`server/crypto_utils.py` 使用 `ADVISOR_MASTER_KEY` 派生 Fernet 密钥。没有主密钥时，不允许加密写入真实敏感字段。

## 当前限制

当前隐私网关是 MVP 级规则系统，还不是完整的数据分级治理中心。下一步应补充记忆分级、用户确认、过期、复审、禁用参与建议和引用追踪。
