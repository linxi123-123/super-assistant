# P3 Case Review 人工评分模板

使用方式：

1. 打开 `http://127.0.0.1:8766/app/history.html`。
2. 逐条点击 15 条 case。
3. 对照页面中的 `phase_context`、`evidence_chain`、`revision_explanation`、`audit_readiness`。
4. 人工填写每条 case 的判断、评分和备注。
5. 不得用系统自评替代人工评分。

评分维度：

- 链路完整性：事件、记忆、假设、信号、触达、反馈、结果、模型修正是否可追踪。
- 判断可解释性：判断是否能被证据链和阶段上下文解释。
- 军师提醒价值：提醒是否对当前阶段和长期目标有真实帮助。
- 反迎合质量：是否敢指出风险，而不是顺着用户扩功能。
- 模型修正合理性：置信度变化和修正理由是否合理。
- 页面可审计性：历史页面是否足以支持人工复核。
