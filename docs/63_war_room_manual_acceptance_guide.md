# W5 个人作战室页面人工验收指南

## 1. 本阶段目标

人工判断个人作战室页面是否真的有“作战室价值”。

重点不是页面好不好看，而是：

- 是否 30 秒内知道当前局势
- 是否 30 秒内知道当前最重要动作
- 是否能看到军师提醒
- 是否能看到反方判断
- 是否能看到不行动后果
- 是否能看到高价值信号
- 是否能看到个人模型假设
- 是否能看到阶段门槛
- 是否能追溯审计入口

## 2. 用户打开地址

`http://127.0.0.1:8766/app/war_room.html`

## 3. 人工验收原则

用户不要因为页面能加载就给高分。

必须判断：

- 它是否像“作战室”
- 它是否帮助用户收敛注意力
- 它是否让用户知道现在不该做什么
- 它是否能提醒风险
- 它是否能体现军师系统，而不是 dashboard

## 4. 通过门槛

只有满足以下条件，才建议进入下一阶段：

- 人工总分 `>= 80/100`
- 当前局势模块 `>= 8/10`
- Advisor Brief `>= 8/10`
- Today's Action `>= 8/10`
- 高价值信号 `>= 8/10`
- 个人模型假设 `>= 8/10`
- 承诺与门槛 `>= 8/10`
- 页面不被认为是普通 dashboard
- 页面不被认为是聊天页
- 页面不被认为是 CRUD 后台
- 用户确认“这个页面对我判断当前状态有帮助”

如果不达标，不允许进入下一阶段，应进入 W5A：作战室价值修正。
