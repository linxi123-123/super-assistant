# 隐私红线与数据分级

生成时间：2026-06-11

## 1. 隐私红线

以下信息默认不得直接发送给外部大模型：

- 真实姓名
- 手机号
- 邮箱
- 身份证/护照
- 具体资产金额
- 持仓数量
- 成本价
- 账户信息
- 公司内部机密
- 私人关系细节
- 健康信息
- 家庭敏感信息
- 详细地理位置
- 合同/财务/法律敏感内容

## 2. 数据分级

### P0 公开信息

例如：

- 公开新闻
- 公开行情
- 公开公告
- 公开网页

可用于外部模型。

### P1 低敏个人偏好

例如：

- 喜欢简短回答
- 关注港股/美股
- 偏好中长期分析

可脱敏后使用。

### P2 中敏个人上下文

例如：

- 当前项目
- 工作计划
- 关注标的
- 目标方向

需要最小化后使用。

### P3 高敏个人数据

例如：

- 持仓
- 成本
- 仓位
- 资产
- 关系
- 公司内部信息

默认不外发，只允许本地使用或强脱敏摘要。

### P4 绝密数据

例如：

- 账户凭证
- 身份证件
- 私密合同
- 密钥
- 未公开商业机密

禁止进入模型调用上下文。

## 3. 脱敏规则示例

原始：

```text
用户持有 0700.HK 腾讯控股，成本 312 港币，仓位 35%。
```

脱敏后：

```text
用户持有一只港股大型互联网平台股，仓位偏重，买入逻辑依赖港股科技情绪和公司基本面修复。
```

## 4. 模型调用前检查

每次调用大模型前必须生成：

- 本次使用了哪些用户数据
- 数据级别
- 是否脱敏
- 是否需要用户授权
- 是否记录审计日志
