# Project Advisor MVP Spec

## 目标

项目军师 MVP 用来回答当前项目方向、阶段、风险和下一步动作，而不是继续生成静态文档感的泛泛总结。

## 当前能力

1. 识别项目类问题
   - 项目、阶段、路线图、下一步、MVP、超级军师、作战室

2. 输出军师式结构
   - 简答
   - 当前阶段判断
   - 主要风险
   - 军师判断
   - 今日唯一动作
   - 不要做
   - 下一步
   - 反方判断

3. 当前默认判断
   - 项目已经从静态作战室转入 FAST-MVP。
   - 最大风险不是页面不够多，而是缺少真实后端、隐私网关和可验证的军师回答。

## 禁止事项

- 不把更多页面当成产品价值。
- 不把文档数量当成进展。
- 不跳到执行代理或自动化操作。
- 不绕过隐私网关。

## 下一步

项目军师 MVP 的下一步不是扩页面，而是持续加厚真实闭环：后端服务、隐私、审计、LLM Gateway、用户画像、市场/项目两个高价值场景。
