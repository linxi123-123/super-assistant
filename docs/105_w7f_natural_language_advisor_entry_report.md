# W7F 自然语言军师入口最小实现报告

生成时间：2026-06-11

## 1. 触发原因

用户反馈：

```text
我在网页端看，我看不懂也找不到我怎么跟军师对话的。
我还要自己手动输入选择什么事件保存，这怎么智能呢？
```

该反馈说明当前网页端虽然有事件录入、决策推演和作战室，但缺少一个符合用户直觉的“跟军师说”入口。

## 2. 本阶段目标

新增最小自然语言入口：

```text
跟军师说
```

用户只需要输入自然语言，系统自动判断：

- 事件类型
- 重要性
- 敏感度
- 证据来源

然后自动跑主循环并生成今日局势。

## 3. 修改文件

- `app/index.html`
- `app/app.js`
- `app/styles.css`
- `tasks/MASTER_ROADMAP.md`

## 4. 实现方式

首页作战室顶部新增“跟军师说”区域。

用户输入一句自然语言后点击：

```text
让军师判断
```

系统会：

1. 自动根据关键词判断事件类型。
2. 自动推断重要性。
3. 自动推断敏感度。
4. 写入当前浏览器本地状态。
5. 运行最小主循环。
6. 自动生成今日局势。

## 5. 智能边界

当前实现是本地启发式规则，不调用外部模型。

它解决的是“用户不需要手动当标注员”的入口问题，不代表已经具备完整自然语言理解能力。
