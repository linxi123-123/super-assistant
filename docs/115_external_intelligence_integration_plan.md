# 外部情报接入规划

生成时间：2026-06-11

## 1. 为什么必须接外部情报

因为用户需要系统回答：

- 今天股市怎么样
- 港股怎么样
- A股怎么样
- 美股怎么样
- 某只股票今天怎么了
- 某个行业发生了什么
- 某个项目外部环境变了吗

这些问题必须依赖实时或准实时数据。

## 2. 第一批外部数据源候选

### 市场类

- A股指数
- 港股指数
- 美股指数
- 个股行情
- 板块表现
- 汇率
- 利率
- 商品

### 新闻类

- 财经新闻
- 公司新闻
- 行业新闻
- 政策新闻
- 公司公告

### 个人工作类

- 日历
- 邮件
- 文档
- 项目管理工具

### 网页搜索类

- 普通网页搜索
- 指定站点搜索
- 研究材料搜索

## 3. 数据接入原则

- 每条数据必须有来源。
- 每条数据必须有时间戳。
- 数据源必须标注可信度。
- 大模型不能编造数据。
- 数据和推断必须分开。
- 高风险领域必须给不确定性提示。

## 4. 第一阶段不接什么

暂不接：

- 券商账户
- 自动交易
- 银行账户
- 健康数据
- 私人聊天记录
- 公司核心机密系统
