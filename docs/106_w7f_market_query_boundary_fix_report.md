# W7F 市场实时问题边界修正报告

生成时间：2026-06-11

## 1. 触发反馈

用户在“跟军师说”中输入：

```text
今天股市怎么样
```

系统没有给出军师视角，而是输出了泛化的项目流程提醒。

## 2. 问题判断

这是一个实时外部信息问题。

当前网页端没有接入实时行情、新闻或外部网络，因此不能回答“今天股市涨跌如何”。

正确军师视角应该是：

- 明确实时数据边界。
- 不编造行情。
- 追问市场范围和用途。
- 要求用户提供可信行情源里的关键材料。
- 再基于用户目标和风险做判断。

## 3. 修正内容

新增本地识别逻辑：

- 识别股市、股票、A 股、美股、港股、大盘、指数、行情等词。
- 识别今天、现在、走势、涨跌、怎么样等时间和行情词。
- 将这类输入标记为需要外部实时信息。

当最新输入属于市场实时问题时，作战室不再输出泛化模型提醒，而是显示：

- 事实边界。
- 军师判断。
- 当前唯一动作。
- 不要做。

## 4. 未触碰事项

本次修正未接入外部网络。

未触碰：

- SQLite。
- JSON 数据。
- npm / 框架。
- 外部情报。
- 执行代理。

## 5. 结论

该修正让军师在面对实时市场问题时先守住事实边界，而不是硬编判断。
