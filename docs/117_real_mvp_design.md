# 真正 MVP：三个高价值闭环

生成时间：2026-06-11

## MVP-1：市场军师

用户问题：

- 今天股市怎么样
- 今天港股/A股/美股怎么样
- 我关注的股票怎么样
- 我持有的股票今天有什么风险

需要：

- 市场行情
- 新闻摘要
- 用户关注列表
- 用户持仓摘要
- 风险偏好
- 军师判断模板

输出：

- A股简况
- 港股简况
- 美股简况
- 关注标的影响
- 持仓风险提醒
- 今日不该做什么
- 下一步看什么

## MVP-2：项目军师

用户问题：

- 当前项目下一步怎么做
- 我是不是跑偏了
- 这个功能该不该继续做
- 现在最重要的任务是什么

需要：

- 项目目标
- 当前阶段
- 任务进度
- 阻塞点
- 历史决策
- 结果反馈

输出：

- 当前阶段判断
- 最大阻塞
- 今日唯一动作
- 不要做清单
- 风险提醒
- 下一步计划

## MVP-3：个人决策军师

用户问题：

- 这件事适不适合我
- 我现在是不是又犯老问题
- 这个机会是不是噪声
- 我该不该投入时间

需要：

- 长期目标
- 承诺记录
- 行为模式
- 过去判断结果
- 当前资源
- 机会成本

输出：

- 决策建议
- 反方判断
- 机会成本
- 行动方案
- 风险边界
