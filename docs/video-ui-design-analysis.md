# Video UI Design Analysis — Super Assistant 个人外脑

## 1. 整体布局

```
┌──────────────────────────────────┐
│  Header / Title bar              │
├──────────────────────────────────┤
│                                  │
│  Chat flow area                  │
│  (气泡式对话，占主屏幕)            │
│  - User bubbles (right-aligned)  │
│  - AI bubbles (left-aligned)     │
│                                  │
│  Welcome state (初始状态)         │
├──────────────────────────────────┤
│  Fixed bottom input bar          │
│  大输入框 + 发送按钮               │
└──────────────────────────────────┘
     ↑ 左侧 sidebar 可选择性滑出
```

无右侧面板。单核心对话流 + 可折叠左侧栏。

## 2. 背景与色调

- **浅色背景**（非暗黑）
- 整体干净、明亮、产品级
- 高留白，减少装饰元素

## 3. 主输入框

- 固定在页面最底部
- 大尺寸输入区域
- 单个发送按钮
- 核心视觉焦点

## 4. AI 回答区域

- **聊天气泡形式**（非卡片，非文章）
- 用户气泡靠右
- AI 气泡靠左
- 气泡有明显区分（颜色/形状）

## 5. 处理中状态

- 自然的产品化表达（非技术日志）
- 可能为打字动画/loading dots

## 6. 侧边栏

- 左侧可选滑出
- 非默认展开
- 包含导航/设置/历史

## 7. 不存在元素（视频中无）

- 无右侧"外脑状态"面板
- 无 job_id 显示
- 无 Worker 状态文字
- 无三栏后台布局
- 无暗黑 control panel 风格

## 8. 字体与间距

- 清晰可读的中文字体
- 适中间距
- 圆角气泡
- 卡片感而非表格感

## 9. 设计决策

基于确认信息，重做的首页将采用：
- 浅色主题 + 聊天气泡流
- 固定底部输入栏
- 可折叠左侧 sidebar
- 无右侧面板
- 技术信息收入折叠区域
- 默认走 Local Claude Worker 链路
