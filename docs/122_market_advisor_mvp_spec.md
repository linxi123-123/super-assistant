# Market Advisor MVP Spec

## 目标

市场军师 MVP 先解决用户真实反馈中的问题：当用户问“今天股市怎么样”时，系统不能输出项目流程话，也不能假装知道实时行情。

## 当前能力

1. 识别市场类问题
   - 股市、股票、A股、港股、美股、行情、大盘
   - 腾讯、英伟达等用户关注标的

2. 基于用户画像回答
   - 读取 `data/user_profile.local.json`
   - 使用关注列表
   - 只在用户确认持仓时使用持仓摘要

3. 实时数据边界
   - 未提供手动行情材料时，明确标记 `no_live_data`
   - 不编造涨跌、指数、新闻或价格
   - 提醒用户补齐指数涨跌、成交量、强弱板块和关注标的变化

4. 手动材料模式
   - 用户粘贴行情或新闻材料时，标记 `manual_context`
   - 只基于用户提供材料做判断

## 输出结构

- 简答
- A股、港股、美股概况
- 关注标的相关性
- 持仓相关性
- 军师判断
- 风险
- 下一步看什么
- 不要做
- 数据状态与时间

## 禁止事项

- 不直接给买卖指令。
- 不说“今天上涨/下跌”除非用户提供可靠材料或后续接入真实数据源。
- 不编造用户持仓。
- 不把单日波动直接升级为长期结论。
