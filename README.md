# 个人超级军师系统

这是一个个人局势智能体项目，不是普通聊天助手。

终局目标是构建一个持续理解用户、维护长期记忆、扫描外部情报、识别关键局势信号、主动触达并逐步具备执行能力的个人超级军师 / Jarvis 系统。

## 核心主循环

```text
感知 -> 事件 -> 记忆 -> 个人模型 -> 外部情报 -> 信号 -> 触达 -> 执行 -> 反馈
```

第一版可以小，但必须保留这个主循环的最小切片。

## 当前阶段

阶段 0：终局方案工程化定稿。

目标是把总纲拆成可开发、可验收、可防走偏的产品规格和任务链路。

## 不可妥协项

- 首页不能以聊天框为中心。
- 记忆必须结构化，不是聊天记录堆积。
- 判断必须有证据链、置信度和反例。
- 外部情报必须关联用户目标，否则视为噪音。
- 系统必须能反驳用户，而不是只迎合用户。
- 执行动作必须绑定目标、完成标准和结果复盘。
- 必须有记忆审计、权限开关和删除机制。
